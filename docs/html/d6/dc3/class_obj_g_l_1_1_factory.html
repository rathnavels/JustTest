Detailed Description

Factory class is an abstract base class that establishes the contract for other derived class that implement the device specific cache and render constructs.

Factory class establishes the blueprint for the implementation using pure virtual functions. These virtual functions are implemented in derived classes that specify the cache and render operations in device specific language constructs.
