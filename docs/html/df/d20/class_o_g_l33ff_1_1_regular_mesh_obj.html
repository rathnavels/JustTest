Detailed Description

OGL33ff::RegularMeshObj class implements the pure virtual functions defined in the abstract RegularMeshObj base class.

OGL33ff::RegularMeshObj sets up the cache and render method as it applies to OpenGL 3.3 fixed function hardware devices.
